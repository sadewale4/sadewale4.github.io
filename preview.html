This page explains how to preview your website locally without pushing to GitHub Pages.

Method 1: Python Simple Server (Recommended)

If you have Python installed, this is the easiest method:

# Navigate to your project directory
cd /path/to/sadewale4.github.io

# Python 3
python -m http.server 8000

# Python 2 (if you have it)
python -m SimpleHTTPServer 8000

Then open your browser and go to: http://localhost:8000

Method 2: Node.js http-server

If you have Node.js installed:

# Install http-server globally
npm install -g http-server

# Navigate to your project directory
cd /path/to/sadewale4.github.io

# Start the server
http-server -p 8000

Then open your browser and go to: http://localhost:8000

Method 3: Live Server Extension (VS Code)

If you're using Visual Studio Code:

  1. Install the "Live Server" extension
  2. Right-click on index.html
  3. Select "Open with Live Server"
  4. Your browser will automatically open to the preview

Method 4: PHP Built-in Server

If you have PHP installed:

# Navigate to your project directory
cd /path/to/sadewale4.github.io

# Start PHP server
php -S localhost:8000

Then open your browser and go to: http://localhost:8000
